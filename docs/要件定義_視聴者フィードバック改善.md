# 要件定義:視聴者フィードバック改善(チャリティーマッチLP)

作成日: 2026-07-15 / 元ネタ: 制作者ではない視聴者としての感想
対象: https://charity.monsterfactory.site/

## 改善1: 「マッチプレー」が分からない

- **問題**: ゴルフを知らない視聴者には対戦形式が伝わらない。企画概要の表に「マッチプレー」とだけ書かれている
- **改善案**:
  - 企画概要の「対戦形式」欄に一言説明を併記する
    - 例: 「マッチプレー(ホールごとに勝ち負けを決め、勝ちホール数で競う方式)」
  - 対決ルールセクションの説明文も「1打差でも勝ちは勝ち。ホールごとの勝負だから最後まで逆転がある」など、面白さが伝わる表現に強化する
- **必要素材**: なし(文言変更のみ)

## 改善2: プレゼント内容を具体的に伝える

- **問題**: クラブを総称で書くと、初見でプレゼント内容が分からない
- **改善案**:
  - サイト全体で「**GENGENが負けたらパター、エンター豊田が負けたらドライバー**」と具体的に表記する
  - 罰ゲーム欄・投票セクションで同じ内容に統一し、応募モーダルは入力に集中できる簡潔な文言にする
- **必要素材**: なし(文言変更のみ)

## 改善3: 企画概要の項目順と罰ゲームの説明位置

- **問題**: 現在の並び(開催日時 → 対戦形式 → 罰ゲーム → チャリティ → 配信予定)だと、罰ゲームが途中に挟まって「?」のまま次の項目を読まされる
- **改善案**:
  - 並びを「**開催日時 → 対戦形式 → 配信予定 → チャリティ → 罰ゲーム**」に変更
  - 罰ゲームを最後に置くことで、直後にある「対決ルール(罰ゲーム詳細)」セクションへ視線がそのまま流れる構成にする(もやもやを次のセクションで即回収)
- **必要素材**: なし(並び替えのみ)

## 改善4: エンター豊田の写真を差し替えたい

- **問題**: 投票セクションのエンター豊田の写真(img/enter.jpg)が気に入らない
- **改善案(いずれか)**:
  - A. 新しい写真素材をもらって差し替え(**推奨・要素材**)
  - B. 対決ポスター(img/match.jpg)からエンター豊田部分を切り出して使用(素材不要だが構図の自由度は低い)
- **必要素材**: 案Aの場合、縦横比が近い写真1枚(正方形〜やや縦長が理想。現在は幅31%・高さ約14remのパネルに表示)
- **確認事項**: どちらの案でいくか。案Aなら写真の支給を待つ

## 改善5: 協賛企業の紹介を「Powered by」風の洗練された表記に

- **問題**: 現在は「協賛企業/スポンサー様のご紹介/本企画を支えてくださるスポンサー様のご紹介です。」と日本語の説明文が続き、野暮ったい
- **改善案**:
  - スタンプ: 「OFFICIAL SPONSOR」
  - 見出し: 「**Powered by GLOBAL HOLDINGS**」
  - 説明文は削除するか、一行だけ小さく「本企画は株式会社グローバル・ホールディングスの協賛でお届けしています。」
  - リンクボタンは「Visit Website」または現行のまま
- **必要素材**: なし(文言・スタイル変更のみ)
- **確認事項**: 英語コピーの最終文言(上記案でOKか)

## 実装順序

1. 改善1〜3+5(文言・並び替え。素材不要なので即実施可能)
2. 改善4(写真素材の支給 or 切り出し案の承認を待って実施)
